PROJECT OVERVIEW
===========================================================

Atlas Agent is a modular AI assistant built using the
smolagents framework.

The goal of Atlas is not to become another chatbot.

The goal is to become an intelligent AI assistant capable of
reasoning, planning, selecting tools, executing actions,
observing results, and solving real-world tasks.

Future versions will support:

• Email Automation
• Calendar Management
• GitHub Assistant
• Slack Integration
• Browser Automation
• Notion Integration
• File Management
• Database Queries
• RAG Knowledge Base
• Multi-Agent Collaboration

===========================================================
DEVELOPMENT PRINCIPLES
===========================================================

1. Keep every tool independent.

2. Test every tool before adding another.

3. Push working code only.

4. Write clean and readable code.

5. Build production-quality architecture.

6. Never hardcode secrets.

7. Store API keys inside .env.

8. Keep functions modular.

9. One responsibility per function.

10. Learn by building.
